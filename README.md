This project has been created as part of the Digital Literacy course. As a Student Digital Ambassador, the aim of this project is to understand and demonstrate the responsible use of digital tools, build a professional online presence, improve communication skills, and develop awareness about cyber safety.

The project consists of five tasks, each focusing on a different aspect of digital literacy.

Task Summary
Task 1 – Digital Literacy Infographic
Created an infographic using Canva to explain digital literacy, useful tools, safe internet practices, professional online presence, and email etiquette.

Task 2 – Digital Portfolio
Created and updated profiles on GitHub, LinkedIn, and Kaggle to build a professional digital presence.

Task 3 – Coding & Collaboration Platforms
Completed a beginner coding challenge on CodeChef and created a Google Form titled “Digital Literacy Awareness Quiz”.

Google Form Link: https://docs.google.com/forms/d/e/1FAIpQLSfpOdBm_3cyBCxdPTFITzuzbQBt3a3nSyravysfBH9E1JSDMQ/viewform?usp=preview

Task 4 – Email Etiquette
Drafted two professional emails and created a checklist of social media do’s and don’ts.

Task 5 – Cybercrime Awareness
Created a UPI / Online Payment Fraud Case Study and a prevention checklist to promote safe online practices.

Conclusion
This project helped me understand that digital literacy is not just about using tools, but also about using them responsibly, safely, and professionally in real-world situations.
